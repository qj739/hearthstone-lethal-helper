## 🔍 常见问题

### Q1: 找不到日志文件？

**A:** 确保：
1. 炉石传说已安装
2. 至少启动过一次游戏
3. 查看输出中的日志路径

### Q2: 手牌识别不全？

**A:** 这是日志解析的固有限制：
- 起手留牌在日志中显示为空
- 抽到的牌会正确显示
- HDT 也有同样限制

### Q3: 斩杀计算不准？

**A:** 检查：
1. 卡牌是否在数据库中
2. 对手是否有嘲讽（会在输出中显示）
3. 运行 `python tests/test_optimal_taunt.py` 验证算法

### Q4: 没有输出？

**A:**
1. 确保已重启炉石传说
2. 检查 log.config 是否安装
3. 进入一局游戏测试

## 🎯 核心亮点

### 1. 完整的嘲讽处理 ⭐⭐⭐

```
对手嘲讽：2/3
我方随从：1/1, 3/3, 5/5

❌ 简单贪心：1+3清嘲讽 → 剩余5 = 5伤
✅ 最优算法：3清嘲讽 → 剩余1+5 = 6伤
```

### 2. 智能法术识别 ⭐⭐

```
手牌：火球术(4费6伤)
法伤：+1（场上有法伤随从）
法力：8/10

计算：6 + 1 = 7伤害，消耗4费
```

### 3. 多维度斩杀 ⭐⭐⭐

```
总伤害 = 场面攻击 + 法术 + 武器 + 冲锋
         ↓          ↓      ↓      ↓
      最优清嘲讽  绕过嘲讽 绕过  绕过
```
