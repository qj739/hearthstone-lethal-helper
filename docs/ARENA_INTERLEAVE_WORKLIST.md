# 攻击穿插清单（随从 + 法术）

> **穿插**：`部分场面随从先攻 → 打出卡牌（战吼/法术）→ 剩余随从再攻/打脸`  
> 不能只在 `spell_first`（先法后攻）或 `attack_first`（先攻后法）首尾施放。  
> 引擎入口：`hdt_python/interleave_board.py` + `lethal_checker._simulate_attack_interleaved_outcome`

## 四、标准用例

| 用例 | 期望场攻 | 备注 |
|------|----------|------|
| 对方 3/5 淤泥，我方 6/6+2/2，手牌疯狂药水 | **3** | 6/6解淤泥→药水偷1/2→2/2+1/2打脸（2+1） |
| 对方 3/5 淤泥+2/2嘲，我方 6/6+2/2，手牌疯狂药水 | **2** | 药水偷2/2→6/6+2/2解淤泥/泥浆，被偷2/2打脸 |

| 对方 2 血嘲，我方 8 攻，手牌邪能弹幕 | **2** | 弹幕1发解嘲，2发打脸（先法后攻也可；穿插用于更复杂场面） |
| 3×4/4+3/5嘲，6/6+1/1，入侵者+混乱打击 | **8** | 战吼穿插（非法术穿插表） |
