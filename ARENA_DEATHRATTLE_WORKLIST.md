# 竞技场亡语随从清单（伤害 / 召唤嘲讽）

> **筛选规则**：HSReplay 竞技场池内 `MINION` + `DEATHRATTLE`；
> 仅保留亡语**造成伤害**（英雄/随从/AOE）或亡语**召唤带嘲讽**随从的卡牌。
> **暂不收录**纯抽牌/发现/回手/洗入等非战斗亡语。
> 网页: [https://hsreplay.net/arena/cards/#view=advanced&tableSort=popularity&cardType=MINION&mechanics=DEATHRATTLE](https://hsreplay.net/arena/cards/#view=advanced&tableSort=popularity&cardType=MINION&mechanics=DEATHRATTLE)
> 数据来源: HSReplay `ArenaTimestampRangeFilter=LAST_4_DAYS`
> 模式: `['ArenaGameTypeFilter.BGT_UNDERGROUND_ARENA', 'ArenaTimestampRangeFilter.LAST_4_DAYS']`
> 卡牌文本: hearthstonejson `zhCN`（`HS_new/json/cards_zhCN.json`）；
> enUS 来源 `HS_new/json/cards.json`
> 竞技场统计: 本地缓存或 HSReplay API（20s 超时）；卡牌 JSON 仅读本地

### 兆示亡语 token 伤害说明

三张卡 **亡语机制完全相同**：随机对一个敌人造成伤害；伤害随本局该 token 线的 **兆示升级次数** 递增：

| 兆示升级 | 亡语随机伤 | 实体 tag |
|---------|------------|----------|
| 0 次（刚召唤） | **2** | `TAG_SCRIPT_DATA_NUM_1=2` |
| 1 次 | **4** | `TAG_SCRIPT_DATA_NUM_1=4` |
| 2 次 | **8** | `TAG_SCRIPT_DATA_NUM_1=8` |

| card_id | 区别 |
|---------|------|
| `CATA_580t` | 拉格纳罗斯的士兵 — 多数兆示战吼/武器召唤 |
| `CATA_150t` | 拉格纳罗斯之手 — 拉格纳罗斯 `CATA_150` 巨型肢 |
| `CATA_150t1` | 同上，变体 card_id（签名皮等），数值一致 |

实现时读场上实体的 `TAG_SCRIPT_DATA_NUM_1` 即为亡语伤害；无 tag 时按 **2** 计。

卡牌 JSON 本地路径: `HS_new/json/cards.json`（enUS）、`HS_new/json/cards_zhCN.json`（zhCN）。
