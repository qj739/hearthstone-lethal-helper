# 清场指向性 / 指向性可打脸法术对照表

供检查斩杀模拟中的**单体目标**法术策略。

## 模拟规则（当前实现）

| 类型 | 有嘲讽 | 无嘲讽 |
|------|--------|--------|
| **清场指向性** | 只点嘲讽随从 | 遍历敌方随从择优（表一全部启用） |
| **指向性可打脸** | 打脸 vs 点嘲讽，取场攻更高 | **只打脸**，不考虑点非嘲讽随从 |
