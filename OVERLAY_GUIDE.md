# 覆盖层窗口使用指南

## 🎮 功能说明

覆盖层窗口会在炉石传说游戏窗口上方显示实时信息：
- ⚔️ 场面攻击力
- ❤️ 我方和对手血量
- 🎯 斩杀检测（总伤害）

## 🚀 使用方法

### 基础使用（白底黑字，推荐）

```bash
python hdt_tracker.py
```

**默认显示：**
- 白色底色，黑色文字
- 显示在炉石窗口左上角
- 始终置顶

### 半透明模式

```bash
python hdt_tracker.py --layered
```

**特点：**
- 透明背景
- 绿色文字
- 更美观，但可能不如白底可见

### 禁用覆盖层（只显示控制台）

```bash
python hdt_tracker.py --no-overlay
```

**适合：**
- 不需要窗口显示
- 只想在控制台看信息
- 测试或调试

## 📊 显示内容

### 示例1: 正常对局

```
场攻: 7
我方: 25+2血 | 对手: 18+0血
总伤害: 13 / 18  (还差5)
```

**说明：**
- 场攻 7 = 我方场面可攻击的随从总攻击力
- 我方 25+2 = 25血 + 2护甲
- 总伤害 13 = 场面 + 法术 + 武器 + 冲锋

### 示例2: 有斩杀

```
场攻: 8
我方: 20+0血 | 对手: 12+0血
⚔️ 斩杀！总伤 14 >= 12 ⚔️
```

**提示：**
- 出现 ⚔️ 符号表示有斩杀
- 控制台也会打印详细伤害来源

### 示例3: 对手有嘲讽

```
场攻: 8 [对手有2个嘲讽]
我方: 22+0血 | 对手: 15+5血
总伤害: 12 / 20  (还差8)
```

**说明：**
- 嘲讽提示出现时，场攻可能被挡住
- 但法术和武器伤害可以绕过嘲讽

## 🎯 窗口位置

覆盖层窗口会：
1. **自动检测炉石窗口** - 通过窗口标题"炉石传说"或"Hearthstone"
2. **自动跟随** - 显示在炉石窗口左上角（偏移20像素）
3. **始终置顶** - 保持在游戏窗口之上

### 如果找不到炉石窗口

显示：
```
未检测到 Hearthstone / 炉石传说 窗口…
```

**解决方案：**
1. 确保炉石传说已启动
2. 检查窗口标题是否正确
3. 如使用英文客户端，修改代码：
   ```python
   overlay = Overlay(title_hint="Hearthstone", ...)
   ```

## ⚙️ 自定义配置

### 修改窗口大小

编辑 `overlay_win.py` 第 183-184 行：

```python
self._fixed_w = 600  # 宽度（像素）
self._fixed_h = 120  # 高度（像素）
```

### 修改透明度

编辑 `hdt_tracker.py` 第 46 行：

```python
self.overlay = Overlay(..., opacity=220)  # 0-255，值越大越不透明
```

### 修改颜色

编辑 `overlay_win.py`：

**白底模式** (第 253 行):
```python
hbrush = gdi32.CreateSolidBrush(RGB(255, 255, 255))  # 白底
gdi32.SetTextColor(hdc, 0x000000)  # 黑字
```

**半透明模式** (第 272 行):
```python
gdi32.SetTextColor(hdc_mem, 0x00FF00)  # 绿字（BGR格式）
```

## 🐛 故障排除

### 问题1: 窗口不显示

**检查：**
1. 炉石传说是否运行
2. 控制台是否有错误信息
3. 尝试白底模式（不加 --layered）

**解决：**
```bash
# 使用白底模式测试
python hdt_tracker.py
```

### 问题2: 窗口位置不对

**原因：**
- 多显示器
- 窗口检测失败

**解决：**
编辑 `overlay_win.py` 第 336 行，修改偏移量：
```python
user32.SetWindowPos(self.hwnd, HWND_TOPMOST,
    rect.left + 20,  # X偏移
    rect.top + 20,   # Y偏移
    ...
)
```

### 问题3: 文字看不清

**白底模式：**
- 增大窗口尺寸
- 修改字体大小（需要修改 GDI 代码）

**半透明模式：**
- 增加透明度：`opacity=255`
- 改用白底模式

### 问题4: 窗口闪烁

**原因：**
- 刷新太频繁

**解决：**
编辑 `hdt_tracker.py` 第 104 行，增加刷新间隔：
```python
if current_time - self.last_print_time < 3.0:  # 改为3秒
```

## 📝 显示内容说明

### 场攻计算

```python
场攻 = sum(能攻击的随从的攻击力)
```

**包括：**
- ✅ 场面上可以攻击的随从
- ✅ 风怒随从计算多次攻击
- ❌ 不包括召唤失调的随从
- ❌ 不包括冰冻/沉默的随从

### 总伤害计算

```python
总伤害 = 场面攻击 + 法术伤害 + 武器伤害 + 冲锋随从
```

**注意：**
- 场面攻击考虑嘲讽（最优解算法）
- 法术伤害自动加法伤
- 法术和武器可以绕过嘲讽

### 血量显示

```
我方: 25+2血
```
= 25生命值 + 2护甲 = 27总血量

## 💡 使用技巧

### 1. 快速判断斩杀

看到 ⚔️ 符号立即知道有斩杀！

### 2. 注意嘲讽提示

```
场攻: 8 [对手有2个嘲讽]
```
如果有这个提示，说明场面攻击可能被挡住，但法术可以打脸。

```
控制台输出：
伤害来源:
  • 场面攻击（清嘲讽后剩余）: 5伤害
  • 法术 CS2_029: 7伤害 (4费)
  • 武器 CS2_091: 3伤害
```

## 🔧 高级配置

### 多显示器支持

如果使用多显示器，窗口会跟随炉石窗口到任意显示器。

### 自动重连

如果炉石窗口关闭后重新打开，覆盖层会自动重新附加。

### 性能优化

覆盖层每 1 秒更新一次（配合日志刷新间隔 2 秒）：
- CPU 占用: < 1%
- 内存占用: < 5MB
- 不影响游戏性能
